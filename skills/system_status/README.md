# System Status Skill

## What it does

Reports a compact runtime status for diagnostics.

## Trigger phrases

- `system status`
- `jarvis status`
- `status report`
- `core version`
- `console version`
- `build datetime`

## Notes

- No environment variables required.
- Response includes OS, Python version, and timestamp.
- Version queries return Core and/or Console version plus build datetime.
